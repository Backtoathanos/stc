Migration plan — why

! 17 active users each get 1 TB of OneDrive storage — most of it currently unused, with important files still sitting on local machines. This plan moves that data to the cloud before the subscription renews, so nothing is lost and storage already being paid for gets used.

Migration flow

1
Audit and notify
Identify users not using OneDrive, inform them of the migration schedule.
2
Login on all devices
Each user signs in to OneDrive on PC, laptop, and mobile with their work account.
3
Move data to OneDrive
Known Folder Move redirects Desktop/Documents/Pictures; remaining files copied manually.
4
File request-response
Admin asks each user which local files/folders matter; user confirms; admin migrates and verifies.
5
Verify and report
Confirm sync status per user, then report completion and storage usage to management.
