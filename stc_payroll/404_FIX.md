# Fixing 404 Error

If you're getting a 404 error, try these solutions:

## Solution 1: Access via public folder (Recommended)

Instead of accessing:
```
http://localhost/stc/stc_payroll/
```

Access directly:
```
http://localhost/stc/stc_payroll/
```

## Solution 2: Update Apache Configuration

If you want to access via `http://localhost/stc/stc_payroll/`, you need to:

1. **Check if mod_rewrite is enabled** in Apache
   - Open `httpd.conf` in XAMPP
   - Make sure this line is uncommented: `LoadModule rewrite_module modules/mod_rewrite.so`

2. **Allow .htaccess overrides**
   - In `httpd.conf`, find the `<Directory>` section for your htdocs
   - Change `AllowOverride None` to `AllowOverride All`

3. **Restart Apache** after making changes

## Solution 3: Create Virtual Host (Best Solution)

Create a virtual host in Apache:

1. Open `C:\xampp\apache\conf\extra\httpd-vhosts.conf`

2. Add this configuration:
```apache
<VirtualHost *:80>
    ServerName stc-payroll.local
    DocumentRoot "E:/xampp/htdocs/stc/stc_payroll/public"
    <Directory "E:/xampp/htdocs/stc/stc_payroll/public">
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

3. Add to `C:\Windows\System32\drivers\etc\hosts`:
```
127.0.0.1    stc-payroll.local
```

4. Restart Apache and access: `http://stc-payroll.local`

## Quick Test

To test if the issue is with .htaccess, try accessing:
```
http://localhost/stc/stc_payroll/index.php
```

If this works, the issue is with the .htaccess rewrite rules.
